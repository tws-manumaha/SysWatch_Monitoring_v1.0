# SysWatch – Simple Monitoring. Smarter Operations.

**SysWatch** is an open‑source infrastructure monitoring platform designed for system administrators who want a tool that is **easy to install, easy to understand, and trustworthy for years**.

It monitors Linux and Windows servers, network devices (routers, switches, printers), and services. It includes:
- **Live dashboard** with host status, trends, and alerts
- **Alert lifecycle** (OPEN → ACKNOWLEDGED → RESOLVED)
- **SSL certificate expiry** warnings (the "grocery store" logic – check once, alert when needed)
- **AI‑driven anomaly detection** (statistical deviation analysis)
- **Email and Microsoft Teams notifications**
- **REST API** for integrations
- **Cross‑platform agent** (Python‑based)
